## Featured projects ⚡

### Lmbic — Interactive Projection System

Spatial-AR passion project: a Jetson + cameras + DLP projector in a
lamp-style form factor, projecting context-aware overlays onto a desk or wall.

- Hardware bring-up, camera–projector calibration with checkerboard homographies
- Closed-loop pipeline with marker tracking, YOLO object detection, and user-masking segmentation
- Latest prototype: Unity + CS:GO Game State Integration projecting ambient
  esports overlays in response to in-game events

**Stack:** Jetson · DLP · OpenCV · YOLO · Unity · C++ · C# · Python
